# Estructura del proyecto
```
Landingpage/
│
├── css/                     # Hojas de estilo
├── img/                     # Imágenes utilizadas en la página
├── js/                      # Scripts JavaScript
├── Wireframe de sitio web.png   # Diseño base / wireframe
└── index.html               # Página principal
```
``
## Cómo visualizar el proyecto
```
1. Clona el repositorio:

git clone https://github.com/brayan-florez/Landingpage.git


2. Abre el archivo index.html en tu navegador.
```
## Características principales
```
* Diseño basado en un wireframe incluido en el repositorio.

* Uso de HTML5 para la estructura.

* Estilos creados con CSS.

* Funcionalidades básicas agregadas con JavaScript.

* Archivos organizados por carpetas para fácil mantenimiento.
```
## Wireframe del sitio
```

El archivo Wireframe de sitio web.png muestra la estructura inicial planeada para la landing page.

```

